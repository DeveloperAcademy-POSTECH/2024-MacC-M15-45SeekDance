## :pushpin: Features

<summary>📌 핵심 기능 </summary>

1. ⛰️ 애플 헬스킷의 계단 오르기 데이터를 통해 내가 얼마나 계단을 많이 올랐는 지 확인하고 해당 레벨에 맞는 보상 아이템을 얻을 수 있습니다.

2. 🪪 계단을 오르며 더 많은 게임 센터 뱃지 성취를 얻을 수 있습니다.

3. 🏆 게임 센터와 연동하여 친구 및 플레이어와 주간 계단 점수 경쟁을 할 수 있습니다.
